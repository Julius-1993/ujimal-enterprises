## What this is
A single-page React + Vite site styled with Tailwind CSS and DaisyUI. Includes SVG demo images and clickable contact buttons that open WhatsApp or your mail client.

## Quick start
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build: `npm run build`

## Important
- Replace the `WHATSAPP_NUMBER` and `EMAIL` values in `src/App.jsx` with your actual contact info.
- The WhatsApp button uses `https://wa.me/<number>`. Use international format (e.g. +2348012345678).
